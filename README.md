 <h1>Endpoints:</h1>
 <ol>
 <li>Get:/users   -  Listar todos usuarios;</li>
<li>Get:/users/:id - Listar único usuário;</li>
<li>Post:/users/ - Criar único usuário;</li>
<li>Put:/users/:id - Alterar único usuário;</li>
<li>Delete/users/:id - Deletar único usuário.</li>
</ol>
